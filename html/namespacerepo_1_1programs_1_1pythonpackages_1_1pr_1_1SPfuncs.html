def repo.programs.pythonpackages.pr.SPfuncs.mgStringHarmonics (   theta,
  x,
  err = None,
  harmonics = [1],
  chi2Filter = inf 
)

Input measurements x taken at angles theta0+ n* dtheta (assumed to be evenly spaced) and return harmonic components.

Returns: phi, amp, err, chi2 where they each have the shape [ (val1, val2), (val1, val2)... ], where each tuple corresponds to a different harmonic and the individual components correspond to the different quadratures. incalculable values are replaced with "nan"

That is, return the coefficients A1, A2 that fit the data for A1*sin(n*theta) + A2*cos(n*theta) for harmonic n, as well as standard deviations and chi^2
